# 🏥 Hospital Management Data Analytics Project

## 📌 Project Overview
This project is a complete Hospital Management Data Analytics system developed using Python, SQL, Excel, SQLite, and Power BI. It helps analyze patient records, appointments, billing information, and hospital performance through interactive dashboards.

---

## 🚀 Tools and Technologies

- Pandas
- SQLite
- SQL
- Excel
- Power BI
- DAX
